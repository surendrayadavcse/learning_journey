HTML & CSS Syntax Guide

1. Basic HTML Structure

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Page</title>
</head>
<body>
    <h1>Welcome to HTML</h1>
</body>
</html>

2. Common HTML Elements

2.1 Headings

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<h3>Heading 3</h3>

2.2 Paragraphs & Links

<p>This is a paragraph.</p>
<a href="https://www.example.com">Click Here</a>

2.3 Lists

<ul>
    <li>Item 1</li>
    <li>Item 2</li>
</ul>

<ol>
    <li>First</li>
    <li>Second</li>
</ol>

2.4 Tables

<table border="1">
    <tr>
        <th>Name</th>
        <th>Age</th>
    </tr>
    <tr>
        <td>John</td>
        <td>25</td>
    </tr>
</table>

2.5 Forms & Inputs

<form action="submit.php" method="POST">
    <label>Name:</label>
    <input type="text" name="name">
    <input type="submit" value="Submit">
</form>

3. Introduction to CSS

3.1 Types of CSS

✔ Inline CSS

Applied directly to an HTML element using the style attribute.

<p style="color: blue; font-size: 20px;">This is inline CSS</p>

✔ Internal CSS

Defined inside a <style> tag within the <head> section.

<style>
    p {
        color: red;
        font-size: 18px;
    }
</style>

✔ External CSS

Stored in a separate file and linked using the <link> tag.

<link rel="stylesheet" href="styles.css">

4. CSS Basics

4.1 CSS Selectors

✔ Element Selector

h1 {
    color: blue;
}

✔ Class Selector

.highlight {
    background-color: yellow;
}

✔ ID Selector

#main {
    font-size: 20px;
}

✔ Grouping Selectors

h1, p {
    color: red;
}

5. Box Model

.box {
    width: 200px;
    height: 100px;
    padding: 10px;
    border: 2px solid black;
    margin: 20px;
}

6. Flexbox Layout

.container {
    display: flex;
    justify-content: center;
    align-items: center;
    height: 100vh;
}
