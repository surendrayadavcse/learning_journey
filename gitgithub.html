Git & GitHub

What is Git?

Git is a version control system that tracks changes in code.

What is GitHub?

GitHub is a cloud-based platform for hosting Git repositories.

Common Git Commands

1. Setup Commands

# Configure Git with your name and email
git config --global user.name "Your Name"
git config --global user.email "your@email.com"

2. Creating and Cloning Repositories

# Initialize a new Git repository
git init

# Clone an existing repository
git clone <repository-url>

3. Staging and Committing

# Add files to staging
git add .

# Commit changes
git commit -m "Your commit message"

4. Branching and Merging

# Create a new branch
git branch new-branch

# Switch to a branch
git checkout new-branch

# Merge branches
git merge branch-name

5. Pushing and Pulling

# Push changes to remote repository
git push origin branch-name

# Pull changes from remote repository
git pull origin branch-name

6. Checking the Status and Logs

# Check repository status
git status

# View commit history
git log
